# 🚀 HirePrep AI – Mock Interview Platform

🌐 Live Demo: https://hireprep-ai-plum.vercel.app  

---

## 🔥 Overview

HirePrep AI is a full-stack AI-powered mock interview platform that simulates real interview scenarios and provides structured feedback to help users improve their performance.

Unlike basic question generators, this platform focuses on **evaluation, personalization, and actionable insights**.

---

## ⚡ Features

- **AI Mock Interviews**
  - Dynamic question generation based on role & difficulty
  - Real interview-like experience

- **Personalized Interviews**
  - Role-based (Frontend, Backend, etc.)
  - Experience-level difficulty adjustment

- **AI Feedback System**
  - Strengths & weaknesses analysis
  - Actionable improvement suggestions

- **Performance Scoring**
  - Overall grade after each interview

- **Interview History**
  - Track and review previous sessions

---

## 🧠 Tech Stack

- **Frontend:** Next.js, Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Database:** PostgreSQL (Neon Serverless)  
- **ORM:** Drizzle ORM  
- **AI Integration:** Gemini API  

---

## 🏗️ Architecture

- Serverless backend using Next.js APIs  
- AI layer for generating questions & feedback  
- PostgreSQL for storing interview sessions  
- Drizzle ORM for type-safe database queries
